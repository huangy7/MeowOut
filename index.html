MeowOut 是一款 macOS 原生菜单栏应用。它通过系统底层接口精准检测用户活跃度，根据连续工作时长自动提醒并引导你规律休息、喝水和呼吸，是每一位开发者最贴心的健康编码伙伴。

✨ 核心特性

为常年久坐、高强度编码的职场人精心定制的健康监控系统

🐱

多样化像素伙伴

内置 Toby (托比橘猫)、Oreo (奥利奥熊猫) 与 Pika (皮卡) 等极简风格宠物，支持实时切换与独立的高清像素动画渲染。

⏳

硬核防沉迷算法

支持时光回滚：精准扣除离座初期的“幽灵时间”，防止计时虚高；满血复活：强制休息阈值确保工作循环得到科学、彻底的重置。

💧

全能喝水提醒

提供“跟随工作节奏”或“自定义时间间隔”双模式。在菜单栏实时以直观百分比或杯数显示饮水进度，保护你的肾脏健康。

🧘

正念呼吸练习

集成 5 分钟呼吸训练模块（支持箱式、4-7-8、腹式呼吸），全屏沉浸式交互引导，练习数据自动同步至健康时间回顾。

📊

现代化管理面板

基于 SwiftUI 构建的侧边栏导航面板，提供深度个性化的设置选项，以及详尽直观的“今日回顾时间轴”和历史统计图表。

⚡️

原生极佳性能

完美适配 macOS 系统。采用 Swift Observation 状态机与 AppKit 多窗口底层协同机制，配合 Canvas 渲染，30FPS 完美兼顾功耗平衡。

📝

Memos 闪念胶囊

不仅是健康伴侣，更是效率利器。支持 Markdown 渲染、时间轴与日历双视图，并集成图片附件管理的原生备忘录。

📋

剪贴板历史

原生级别的轻量级剪贴板管理工具。支持纯文本、富文本与图片，提供极速的搜索、预览与快捷键操作，让你的复制粘贴效率翻倍。

🔐

Toolbox 2FA

内置安全可靠的双因素验证器，支持多哈希算法，依托 Keychain 提供金融级加密存储。

📁

原生文件中转站

跨应用文件拖拽的中转枢纽。支持图片、文本、任意文件格式临时寄存与快速分发，彻底告别频繁切换窗口的烦恼。

💻

合盖防休眠

支持自定义外接显示器合盖不休眠机制，并集成了智能电池保护，全方位提升 Mac 工作流体验并延长硬件寿命。

🍺 使用 Homebrew 安装 (推荐)

如果你的 Mac 上已安装 Homebrew，可以通过我们的专属 Tap 一键安装和更新：

brew install huangy7/tap/meowout

⚡️ 极速一键安装

打开终端，复制并粘贴运行以下一键脚本即可快速安装并打开 MeowOut：

curl -L https://github.com/huangy7/MeowOut/releases/latest/download/MeowOut.dmg -o ~/Downloads/MeowOut.dmg && xattr -cr ~/Downloads/MeowOut.dmg && open ~/Downloads/MeowOut.dmg

或手动下载对应芯片的最新 Stable 安装包：

 Apple Silicon (M1/M2/M3) MeowOut-1.3.3-arm64.dmg Intel 处理器版 MeowOut-1.3.3-x86_64.dmg

⚠️ 注：若首次运行提示“无法打开/未识别开发者”，请在终端运行 `xattr -cr /Applications/MeowOut.app` 即可解锁。
