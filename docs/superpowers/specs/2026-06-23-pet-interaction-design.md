# MeowOut: Pet Edge Snapping Design

## 1. Goal
在最小化系统开销的前提下，增加“边缘半隐藏吸附”机制。允许用户手动将宠物拖拽到屏幕左右边缘，宠物会自动吸附并“藏起半个身子”，方便用户在专注时减少视觉干扰，同时保留随时唤醒的能力。

## 2. Scope & Features

### 2.1 手动拖拽与“探半个身子”吸附 (Half-hidden Edge Snapping)
- **触发条件**：用户用鼠标拖拽宠物，当宠物中心点靠近屏幕左侧或右侧边缘（如距离边缘 < 30pt）时释放鼠标。
- **半隐藏吸附**：释放鼠标后，宠物会平滑地向边缘滑行，并且**将自己一半的身子藏到屏幕外**（即宠物的中心点对齐到屏幕的边界坐标）。这样在屏幕上看起来就像宠物在边缘探出了半个脑袋暗中观察，类似 360 悬浮球的收纳效果。
- **状态切换**：吸附后，宠物停止全屏巡逻，进入静态陪伴状态。
- **唤醒恢复**：用户只需再次按住并往屏幕中心拖拽，宠物即可脱离边缘，恢复完整的全貌和巡逻逻辑。

## 3. Architecture & Dependencies
- **坐标偏移计算**：在计算吸附坐标时，不再是 `x = 宠物宽度/2`，而是直接设置为 `x = 0`（左边界）或 `x = screenWidth`（右边界），利用 macOS 窗口机制自然隐藏超出的部分。
- **拖拽事件判定**：在 `CatOverlayController.swift` 或处理鼠标释放逻辑的地方，检测拖拽结束的最终坐标。
- **动画平滑**：使用状态的渐变使得从拖拽结束点到吸附点有一段丝滑的坐标位移动画，而不是瞬间闪现。

## 4. Out of Scope (已排除)
- 🚫 不做全屏状态自动监听与隐藏（保持性能极致轻量）。
- 🚫 不做全局光标摇晃捕捉，避免多余的系统事件注册。
